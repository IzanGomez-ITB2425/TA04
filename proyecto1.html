Proyecto 1: Obtención y Gestión de Datos de Incidencias

Descripción detallada de los pasos y resultados de mi proyecto para gestionar incidencias de equipos informáticos y audiovisuales.

1. Crear un Formulario para Recoger Datos

El primer paso del proyecto consistió en diseñar un formulario que permitiera a los usuarios reportar incidencias sobre equipos informáticos y audiovisuales dentro de una organización.

El formulario recoge los siguientes datos:

  • Nombre del equipo
  • Tipo de incidente (hardware, software, etc.)
  • Descripción detallada del problema
  • Fecha del incidente
  • Nombre del reportero

Los datos recogidos se enviaron a Google Forms y se almacenaron en una hoja de cálculo de Google Sheets, desde donde se exportaron a un archivo CSV para su posterior procesamiento.

2. Migración de los Datos de CSV a XML

Una vez que los datos fueron recolectados y almacenados en formato CSV, se realizó una conversión a formato XML. Esto se hizo para estructurar mejor la información y facilitar su posterior análisis.

El archivo CSV (incidencias.csv) fue convertido al archivo XML (incidencias.xml), lo que permitió organizar los datos en un formato más adecuado para su procesamiento y análisis automatizado.

3. Procesamiento de los Datos y Visualización en Pantalla

En este paso, se desarrolló un programa en Python que leía los datos del archivo XML y los procesaba para mostrar información útil. Se creó un filtro para que el programa identificara incidencias relevantes y descartara aquellas que no aportaran valor (como datos de incidencias futuras, p. ej. 2064).

El programa mostraba los datos de manera formateada, con colores y tabuladores, para facilitar su lectura y comprensión.

4. Almacenamiento de la Información en Formato JSON

Finalmente, los datos procesados fueron almacenados en formato JSON. Este paso añadió una nueva funcionalidad al programa en Python, permitiendo guardar los resultados en un archivo incidencias.json.

Este archivo JSON puede ser utilizado en aplicaciones futuras que requieran acceder y procesar los datos de forma eficiente.
